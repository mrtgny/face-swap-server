# face-swap-server
Face Swap Server

```
git clone https://github.com/mrtgny/face-swap-server.git
```

```
cd face-swap-server
```

```
npm install
```

```
cd faceSwapPython
```

```
./setup_env.sh
```

## To check the script you can use

```
./face_swap.sh
```

## Before
```npm start```
You must change the IP address in the

```
./bin/www
```

with your local IP
